# Manu.Blazor.Components

### Add refeference in Index.html or _Host.cshtml
```
<link href="_content/Manu.Blazor.Components.styles.css"/>
```

### Example
``` html
@page "/"
...
<!DOCTYPE html>
<html lang="en">
<head>
    ...
    <link href="_content/Manu.Blazor.Components.styles.css"/>
    ...
</head>
<body>
    ...
    <script src="_framework/blazor.server.js"></script>
</body>
</html>
```
